# Key Concepts
## Basic Authentication

1. Username and password sent in request header
2. Simple but not very secure

## Token Authentication

1. Server generates a token after login
2. Token must be sent with every request

## JWT Authentication
1. Uses signed JSON tokens
2. Stateless authentication
3. Widely used in modern APIs
