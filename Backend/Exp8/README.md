# Aim:
A simple RESTful API built using Flask that performs CRUD operations for managing student records. This project demonstrates modular Flask application structure using Blueprints.

# Features
1. Create a new student
2. Retrieve all students
3. Retrieve a single student by ID
4. Update student details
5. Delete a student
6. Modular structure using Flask Blueprints
7. JSON-based request and response handling

# Testing
You can test the API using:
1. Postman
2. Thunder Client (VS Code)
3. cURL

# Technologies Used

1. Python
2. Flask
3. Postman (for API Testing)

# Learning Outcomes
1. Understand Flask application structure
2. Implement RESTful API design
3. Work with JSON data
